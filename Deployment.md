# Cafe Flow Management System - Deployment Guide

## 📌 Project Overview
This project is developed in C# Windows Forms with database integration. It manages cafe operations like orders, products, users, and billing.

---

## ⚙️ Requirements
- Visual Studio 2019/2022
- .NET Framework 4.5 or higher
- MySQL or SQL Server
- Git (optional)

---

## 🚀 How to Run

### Step 1: Clone Repository
git clone https://github.com/sa8385123-creator/Cafe-Flow-Management-System.git

### Step 2: Open Project
- Open Visual Studio
- File → Open → Project/Solution
- Select `.sln` file

### Step 3: Run Project
- Press Start (F5)
